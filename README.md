# Java DSA Practice

This repository contains Java implementations of various Data Structures and Algorithms problems.

# ⭐ **PATTERN 1: Basic Math & Logic (Very Common in TCS, Infosys)**

**Concepts**

* Prime numbers
* Fibonacci
* Palindrome
* Armstrong numbers
* GCD / LCM
* Factorial
* Sum of digits / reverse digits

**Questions**

1. Check if a number is prime
2. Print first N Fibonacci numbers
3. Check if a number is palindrome
4. Find GCD of two numbers
5. Find LCM of two numbers
6. Armstrong number check
7. Count digits in a number
8. Reverse an integer
9. Sum of digits
10. Strong number check
